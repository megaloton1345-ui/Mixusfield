Getting Started

Installation

Mixusfield is loaded via a single loadstring call. No manual file management needed.

lua 
local Mixusfield = loadstring(game:HttpGet(
    "https://raw.githubusercontent.com/LumeCraftors01/Mixusfield/refs/heads/main/Mixusfield.lua.txt
"
))()
ℹ
HttpService Required Make sure your game or executor has HttpEnabled. In Roblox Studio go to Game Settings → Security → Allow HTTP Requests.
Getting Started

Quick Start

Create a window with tabs and components in under 20 lines.

lua — full example 
-- 1. Load the framework
local Mixusfield = loadstring(game:HttpGet("https://raw.githubusercontent.com/LumeCraftors01/Mixusfield/refs/heads/main/Mixusfield.lua.txt"))()

-- 2. Create a window
local Window = Mixusfield:CreateWindow({
    Name     = "My Script",
    Subtitle = "v1.0",
    Theme    = "Dark",
    Accent   = Color3.fromRGB(124, 107, 240),
    ToggleButton = { Enabled = true },
})

-- 3. Add a tab
local Main = Window:CreateTab({
    Name = "Main",
    Icon = { Type = "Lucide", Source = "home" },
})

-- 4. Add a button
Main:CreateButton({
    Name     = "Click Me!",
    Callback = function()
        Mixusfield:Notify({ Title="Hello!", Content="Button clicked.", Type="success" })
    end,
})
Core

CreateWindow

The root of every UI. Creates a draggable, minimizable, resizable window with an optional floating toggle button.

lua 
local Window = Mixusfield:CreateWindow({
    Name        = "Mixusfield",       -- window title
    Subtitle    = "v2.0.0",          -- shown below title
    Theme       = "Dark",            -- "Dark"|"Light"|"AMOLED"|"Midnight"|"Rose"|"Ocean"
    Accent      = Color3.fromRGB(124,107,240),  -- optional accent
    MinSize     = { Width=360, Height=260 },  -- optional
    ToggleButton = {
        Enabled = true,
        Icon    = { Type="Lucide", Source="menu" },
    },
    KeySystem   = {                   -- optional key gate
        Title    = "Key Required",
        Key      = "MY_KEY_HERE",
        SaveKey  = true,
        Gateways = {
            { Name="Get Key", URL="https://..." },
        },
    },
})
ParameterTypeDescription
NamereqstringWindow title shown in the top bar.
SubtitleoptstringSmaller text displayed below the title.
ThemeoptstringOne of: "Dark" "Light" "AMOLED" "Midnight" "Rose" "Ocean". Default: "Dark".
AccentoptColor3Override the theme's accent color.
MinSizeopttable{ Width, Height } — minimum resize dimensions.
ToggleButtonopttableFloating button to show/hide the window. Set Enabled = true to activate.
KeySystemopttableKey gate before showing UI. See Key System.

Components

Component Library

All components are added to a Tab object. Each returns a handle with :Get() and :Set() methods.

Button interactive

Clickable action with optional icon, tooltip, and ripple effect.

No state handle returned.

Toggle interactive

On/off boolean switch with spring animation.

toggle:Get() → bool
toggle:Set(bool)

Slider interactive

Draggable numeric range input with suffix support.

slider:Get() → number
slider:Set(number)

Dropdown interactive

Option picker with search and live refresh.

dd:Get() → string
dd:Set(str) · dd:Refresh(tbl)

Textbox input

Free-text input with placeholder support.

tb:Get() → string
tb:Set(str)

Keybind input

Key assignment widget — click to record a keypress.

kb:Get() → KeyCode
kb:Set(KeyCode)

ColorPicker input

HSV color picker with hex display.

cp:Get() → Color3
cp:Set(Color3)

ProgressBar display

0–100 animated progress indicator.

pb:Get() → number
pb:Set(0–100)

RadioGroup interactive

Mutually exclusive option selector.

rg:Get() → string
rg:Set(str)

Accordion layout

Collapsible group that can contain any other components.

Returns (accordion, innerTab)

    end,

Warning

✕

Mixusfield:Notify({
    Title    = "Done!",
    Content  = "Script executed successfully.",
    Type     = "success",  -- "success" | "warning" | "error" | "info"
    Duration = 4,           -- seconds (optional, default 4)
    Icon     = { Type="Lucide", Source="check" },  -- optional

Alert / Confirm Dialogs

Modal dialogs for warnings and confirmations. Blocks the UI until dismissed.

lua
-- Simple alert
Mixusfield:Alert({
    Title   = "Notice",
    Content = "Please read the rules before continuing.",
    Type    = "info",
})

-- Confirm dialog with callbacks
Mixusfield:Alert({
    Title     = "Reset Settings",
    Content   = "This will reset all settings. Are you sure?",
    Type      = "warning",
    Confirm   = "Reset",
    Cancel    = "Cancel",
    OnConfirm = function()
        Mixusfield:Notify({ Title="Reset", Content="Settings cleared.", Type="info" })
    end,
    OnCancel  = function()
        print("Cancelled.")
    end,
})
ParameterTypeDescription
Typeoptstring"info" "warning" "error" "success"
ConfirmoptstringLabel for the confirm button. If absent, only a dismiss button shows.
CanceloptstringLabel for the cancel button.
OnConfirmoptfunctionFires when user confirms.
OnCanceloptfunctionFires when user cancels or closes.

Key System

Gate your UI behind a key. Supports saving the key to DataStore and multiple gateway links.

lua
local Window = Mixusfield:CreateWindow({
    Name      = "My Script",
    KeySystem = {
        Title    = "Key Required",
        Key      = "FREE_2025",
        SaveKey  = true,       -- remember valid key across sessions
        Gateways = {
            { Name="Get Key",     URL="https://linkvertise.com/..." },
            { Name="Backup Link", URL="https://loot-link.com/..."    },
        },
    },
})
✓
SaveKey = true When enabled, a validated key is saved so the user only needs to enter it once. Subsequent loads skip the key screen automatically.

Mixusfield has a built-in search bar. Components registered in the search registry appear in real-time filter results.

Automatic All components are automatically registered in the search system. No extra code is needed — the search bar appears in the window header by default.
API Reference

Framework API

Top-level methods on the Mixusfield object.

Mixusfield:CreateWindow(config: table)→ Window
Creates and returns a new window. See Window section for all config options.
Mixusfield:Notify(config: table)
Sends a notification. Fields: Title, Content, Type ("success"/"warning"/"error"/"info"), Duration, Icon.
Mixusfield:Alert(config: table)
Shows a modal dialog. See Alerts section for options.
Mixusfield:SetTheme(name: string)
Switches the active theme live. Valid values: "Dark" "Light" "AMOLED" "Midnight" "Rose" "Ocean".
Mixusfield:Destroy()
Destroys all windows, cleans up all connections and GUI instances. Call this when unloading your script.
API Reference

Window API

Methods on the Window object returned by CreateWindow.

Window:CreateTab(config: table)→ Tab
Creates a tab. Receives Name and optional Icon.
Window:Show() / Window:Hide()
Show or hide the window with a tween animation.
Window:Toggle()
Toggle visibility.
Window:Minimize() / Window:Restore()
Minimize to top bar or restore to full size.
Window:Notify(config: table)
Same as Mixusfield:Notify but scoped to this window.
Window:Destroy()
Destroys only this window and cleans up its connections.
